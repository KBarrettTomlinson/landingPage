RECOVREE


Recovree is a full-stack web application which utilizes technology to support addiction treatment and aftercare. It was a four person team project.

Composed of a daily survey, data visualization, SMS reminders and alerts, and administrative access to anonymized aggregated data, Recovree aims to ease the abrupt transition which typically follows treatment.
